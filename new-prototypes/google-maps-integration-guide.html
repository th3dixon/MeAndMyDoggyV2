Google Maps Integration Guide

1. Get Google Maps API Key

  1. Go to Google Cloud Console
  2. Create a new project or select existing
  3. Enable "Maps JavaScript API"
  4. Create credentials → API Key
  5. Restrict API key to your domain for security

2. Add to Your HTML

<!-- Replace YOUR_API_KEY with actual key -->
<script async defer
    src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap">
</script>

3. Initialize Map Function

function initMap() {
    // Center on London by default
    const mapOptions = {
        center: { lat: 51.5074, lng: -0.1278 },
        zoom: 12,
        styles: [
            {
                featureType: "poi",
                elementType: "labels",
                stylers: [{ visibility: "off" }]
            }
        ]
    };
    
    const map = new google.maps.Map(
        document.getElementById('map'), 
        mapOptions
    );
    
    // Add provider markers
    providers.forEach(provider => {
        const marker = new google.maps.Marker({
            position: { 
                lat: provider.latitude, 
                lng: provider.longitude 
            },
            map: map,
            title: provider.name,
            icon: provider.isPremium ? {
                url: 'gold-marker.png',
                scaledSize: new google.maps.Size(40, 40)
            } : {
                url: 'blue-marker.png',
                scaledSize: new google.maps.Size(30, 30)
            }
        });
        
        // Info window
        const infoWindow = new google.maps.InfoWindow({
            content: `
                <div class="p-3">
                    <h3 class="font-bold">${provider.name}</h3>
                    <p>★ ${provider.rating} • ${provider.distance}</p>
                    <button class="mt-2 px-3 py-1 bg-orange-500 text-white rounded">
                        View Details
                    </button>
                </div>
            `
        });
        
        marker.addListener('click', () => {
            infoWindow.open(map, marker);
        });
    });
}

4. Custom Marker Icons

Create custom SVG markers for premium and regular providers:

// Premium marker (gold)
const premiumIcon = {
    path: google.maps.SymbolPath.CIRCLE,
    fillColor: '#F59E0B',
    fillOpacity: 1,
    strokeColor: '#D97706',
    strokeWeight: 2,
    scale: 12
};

// Regular marker (blue)
const regularIcon = {
    path: google.maps.SymbolPath.CIRCLE,
    fillColor: '#3B82F6',
    fillOpacity: 1,
    strokeColor: '#2563EB',
    strokeWeight: 2,
    scale: 10
};

5. Marker Clustering

For many providers, use MarkerClusterer:

<!-- Add MarkerClusterer library -->
<script src="https://unpkg.com/@googlemaps/markerclusterer/dist/index.min.js"></script>

<script>
// Create marker clusterer
const markers = providers.map(provider => {
    return new google.maps.Marker({
        position: { lat: provider.lat, lng: provider.lng },
        map: map
    });
});

new markerClusterer.MarkerClusterer({ 
    markers, 
    map,
    algorithm: new markerClusterer.GridAlgorithm({ maxDistance: 40 })
});
</script>
